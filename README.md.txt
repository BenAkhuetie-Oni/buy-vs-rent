# Buy vs Rent Calculator (GitHub Pages)

Static, client-side Buy vs Rent calculator using:

Net Worth Impact (NWI) = Equity − Out-of-pocket costs

## Run locally
Just open `index.html` in a browser.

## Deploy to GitHub Pages
1. Create a GitHub repo (public).
2. Upload `index.html`, `styles.css`, `script.js`.
3. Go to **Settings → Pages**
4. Source: **Deploy from a branch**
5. Branch: **main** / folder: **/(root)**
6. Save, then your site will be live at:
   `https://YOUR-USERNAME.github.io/YOUR-REPO/`
